# Analysis API

This section documents ReaxKit analysis tasks in the current task-based API layout.
Each page describes request/result models and task behavior for one analysis module.

## Structure

Analysis docs are grouped by domain:

- `active_sites`
- `connectivity`
- `control`
- `electrostatics`
- `force_field`
- `kinematics`
- `molecular_analysis`
- `params`
- `timeseries`
- `trajectory`
