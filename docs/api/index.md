# API Overview

ReaxKit API documentation is organized into four sections:

- [Analysis API](analysis/index.md): task request/run/result contracts and analysis outputs.
- [Engine API](engine/index.md): parsers, generators, adapters, and engine-facing I/O.
- [Utils API](utils/index.md): reusable utility functions and numerical helpers.
- [Workflows API](workflows/index.md): CLI workflows and orchestration layers.

Use this page as the top-level entry point when navigating internal APIs.
