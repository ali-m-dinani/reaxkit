# Tutorials

This section contains step-by-step tutorials for the current ReaxKit CLI
structure. If you are new, follow them in order.

---

## Tutorial sequence

1. **[01 - Understanding the Quick Start](01_understanding_quickstart.md)**  
   Explains the current command flow and core concepts such as `frame`, `iter`,
   and `time`.

2. **[02 - Atom-Property Plots and Video Workflows](02_atom_property_and_video_workflows.md)**  
   Uses `plot_atom_property` for per-frame spatial visualization and `gen-video`
   to build animations from saved frames.

3. **[03 - Generating Electric-Field Schedules (`eregime.in`)](03_eregime_generator.md)**  
   Covers `gen_eregime` profiles (`sin`, `pulse`, `func`) and their key
   parameters.

4. **[04 - Meta Plotting with `gen-plot`](04_gen_plot_workflow.md)**  
   Shows file-agnostic plotting from tabular data (`single`, `directed`, `dual`,
   `tornado`, `scatter3d`, `heatmap2d`).
